收藏夹

By qazwsxedc121

5月 16 2014 更新日期:6月 28 2014

此处放一些时不时需要重新找的东西

命令行工具

  • tldr 简化版的man,方便阅读
  • ledger 命令行记账工具
  • tpp 命令行PPT
  • ack 为programmer增强的grep
  • zsh z-shell
  • oh-my-zsh zsh默认增强版配置
  • powerline vim & 命令行界面美化+加强
  • shadowsocks 轻量级socks代理

terminal emulator

  • finalterm 下一代term,各种高级功能
  • iterm2 常用term

编辑器(new)

  • atom github出品编辑器,基于本地node-webkit
  • neovim 下一代vim

Web Frameworks(UI)

  • bootstrap
  • foundation 类似于bootstrap,有不少新的布局
  • ionic 移动端的app framework,类似于jquery mobile

web前端:Slideshow

  • impress.js类似于prezi的展示
  • reveal.js更接近传统PPT,但是有层级概念
  • deck.js与上面类似,注重一步步显示而不是页面切换
  • fullpage.js全页面展示用插件,不限于PPT

web前端:Rich Text editor

  • Sir Trevor新概念富文本编辑器,以block为单位编辑
  • ueditor百度出的富文本编辑器

Web前端:构建

  • yeoman构建脚手架工具,快速生成项目框架
  • grunt自动化构建工具
  • glup同样是自动化构建工具,但是用的方式不同(streaming)
  • bower前端用的包管理器,类似于npm但是用于frontend

web前端:表单相关

  • multi-select 双栏式多选插件
  • Parsley.js 表单验证插件
  • bootstrap-datetimepicker
  • chosen 增强型下拉框
  • jQuery-File-Upload
  • datetimepicker 基于jquery
  • Jcrop 简单的图片裁剪
  • MotionCAPTCHA 在canvas画画来做验证码

JavaScript

  • underscoreJS通用函数式编程库,提供例如map,filter等方便使用的函数(有一个小template engine)
  • lodash underscore的增强版,附加了一些功能,可以替换underscore

Misc

  • 标点符号怎么用英语说
